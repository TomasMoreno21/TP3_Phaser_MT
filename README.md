# SALVAVIDAS

**Está mal, pero no tan mal**

## Nombre y apellido
Tomás Moreno

## Descripción breve
Sos un bombero en un edificio en llamas. Tu misión es empujar a los civiles fuera de las zonas de peligro antes de que los aplasten los escombros que caen. A simple vista empujar está mal, pero en este contexto es la única forma de salvar vidas.

## Explicación del concepto "Está mal, pero no tan mal"
Empujar a una persona está mal. Sin embargo, en este juego el bombero empuja a los civiles para sacarlos de zonas donde están por caer escombros. La acción sigue siendo brusca, pero está justificada porque sin ella los NPCs resultarían heridos.

## Instrucciones para ejecutar el juego localmente
1. Clonar el repositorio
2. Abrir la carpeta del proyecto en VS Code
3. Usar Live Server o cualquier servidor HTTP estático para servir el `index.html`
4. También funciona abriendo el archivo `index.html` directamente en el navegador
